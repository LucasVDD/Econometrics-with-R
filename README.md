# Econometrics with R

This repository provides a structured introduction to econometric theory in R.
The content is designed to be self contained where the only prerequisites needed are introductory courses in statistics, linear algebra and calculus.
It is recommended to start with the first two sections and after that you can jump to whatever topic that interests you the most. 
If you have already taken a course in Econometrics you may skip them and start wherever.
